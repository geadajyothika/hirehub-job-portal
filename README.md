# HireHub Job Portal

A Job Portal website built using HTML, CSS, and JavaScript.

## Features
- Home Page
- Login Page
- Registration Page
- Job Listings
- Apply for Jobs
- Recruiter Dashboard
- Form Validation
- Local Storage

## Technologies Used
- HTML
- CSS
- JavaScript
